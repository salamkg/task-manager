## Task Manager

- git clone
- go to the project directory
- install Sail into the project: _composer require laravel/sail --dev_
- run sail install: _php artisan sail:install_
- start sail: _./vendor/bin/sail up_
- run migrations
- create super-user with _sail artisan register:super-admin_
- enjoy the project ;-)
